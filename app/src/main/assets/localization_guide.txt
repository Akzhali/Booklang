# Руководство по локализации приложения BookLang

## Завершенные шаги

1. Созданы языковые ресурсы в директориях:
   - `app/src/main/res/values/strings.xml` (казахский язык, по умолчанию)
   - `app/src/main/res/values-ru/strings.xml` (русский язык)
   - `app/src/main/res/values-en/strings.xml` (английский язык)
   - `app/src/main/res/values-kk/strings.xml` (дополнительный казахский язык)

2. Реализован класс `LanguageUtil` для управления локализацией:
   - Сохранение выбранного языка в SharedPreferences
   - Переключение языка во время работы приложения
   - Определение текущего языка

3. Обновлен класс `MyApplication` для применения языка при запуске:
   - Добавлен метод `attachBaseContext` для установки локали приложения

4. Обновлен класс `MainActivity` для поддержки локализации:
   - Добавлен метод `attachBaseContext` для поддержки языковых настроек
   - Добавлен метод `applyOverrideConfiguration` для корректной работы на разных версиях Android

5. Обновлен `ProfileFragment` для поддержки смены языка:
   - Добавлено диалоговое окно выбора языка с поддержкой всех локализаций
   - Добавлен механизм перезапуска активности для применения изменений
   - Обновлены строковые ресурсы в layout-файле

## Шаги для завершения локализации

1. Обновить XML-файлы всех остальных макетов, заменив жестко закодированные строки на ссылки на строковые ресурсы:
   - `app/src/main/res/layout/fragment_home.xml`
   - `app/src/main/res/layout/fragment_library.xml`
   - и другие макеты

2. Обновить все остальные фрагменты для использования строковых ресурсов в коде:
   - Заменить жестко закодированные строки на `getString(R.string.key)`
   - Использовать строковые ресурсы в Toast-сообщениях
   - Использовать строковые ресурсы в диалоговых окнах

3. Проверить все активности, чтобы убедиться, что они правильно применяют языковые настройки

## Тестирование локализации

1. Запустите приложение и перейдите на экран профиля
2. Выберите пункт "Тіл" и поменяйте язык
3. Проверьте, что после перезапуска приложения весь интерфейс отображается на выбранном языке
4. Повторите для всех поддерживаемых языков

## Дополнительные советы

1. Для добавления новых строк в приложение всегда сначала добавляйте их в `strings.xml`, а затем обновляйте аналогичные файлы для других языков.
2. При изменении строк убедитесь, что все языковые версии обновлены соответствующим образом.
3. Тестируйте приложение на различных устройствах и версиях Android, чтобы убедиться, что локализация работает корректно.
